# DDD
A Dynamic Deception–Driven (DDD) Framework for Intelligent Proactive Cyber Defens
# DDD-SOAR Replication Package

This repository contains the replication package for the thesis:
**"دفاع پیش‌دستانه هوشمند در امنیت سایبری با استفاده از تله‌گذاری پویا"**
(DDD-SOAR: Direct-to-Dynamic Deception SOAR).

## Contents
- `data/` : Simulation outputs used to generate the reported results.
- `code/` : Scripts for running simulations and/or reproducing analyses.
- `docs/` : Documentation (scenario definitions, metrics, and dataset description).

## How to use (minimal)
1) Review `docs/DATASET.md` for file descriptions, columns, and metrics.
2) If analysis scripts are provided, run them from `code/` (see `code/README.md`).

## Reproducibility notes
This repository is intended to support transparency and reproducibility of the thesis experiments.
All key scenarios (e.g., Brute Force, SQL Injection, Lateral Movement) and metrics are documented in `docs/`.
